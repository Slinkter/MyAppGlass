# MyAppGlass

Sitio web comercial para Glass & Aluminum Company S.A.C.

## Instalación

```bash
pnpm install
cd functions && pnpm install
```

## Desarrollo

```bash
pnpm dev
```

## Build y Deploy

```bash
pnpm run build
firebase deploy
```

## Estructura

-   `src/components`: Componentes reutilizables
-   `src/data`: Datos centralizados
-   `src/layout`: Layouts y navegación
-   `src/routes`: Vistas principales

## Variables de entorno

Configura `.env` con tus credenciales de Firebase.

## Optimización y Clean Code

-   Uso de custom hooks para lógica compartida
-   Componentes documentados con JSDoc
-   Imágenes optimizadas y carga diferida
-   Lint limpio y sin warnings
